# ERC-721 NFT CLI 实现计划

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development or superpowers:executing-plans to implement this plan.

**Goal:** 实现一个 Go CLI 工具，用于以太坊 ERC-721 NFT 的钱包管理、图片/metadata 上传、合约部署和批量铸造。

**Architecture:** cobra CLI 框架 + internal 包分层架构。cmd/ 定义命令，internal/ 实现业务逻辑。合约 ABI/Bytecode 内嵌到 Go 代码中。

**Tech Stack:** Go, cobra, go-ethereum, Solidity, Hardhat, Pinata API

---
